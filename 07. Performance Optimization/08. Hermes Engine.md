**Hermes** is a lightweight JavaScript engine optimized for React Native, developed by Meta.
It is designed to improve app performance, reduce memory usage, and speed up app launch time, especially on Android.

---

## ðŸ”¹ Why React Native Needed Hermes

Traditional React Native apps used JavaScript engines like **JavaScriptCore**, which:

- Parse JS at runtime
- Increase startup time
- Consume more memory

**Hermes solves these problems by precompiling JavaScript into bytecode.**

## ðŸ”¹ Common Interview Question Answers

**Q: Why is Hermes faster than JavaScriptCore?**

> Because Hermes uses **ahead-of-time compilation (AOT)** and runs bytecode instead of raw JavaScript.

**Q: When should you disable Hermes?**

> Rarely. Only if a native SDK or third-party library strictly depends on JavaScriptCore-specific features.

**Q: Does Hermes affect Expo?**

> Yes. Modern Expo projects also support Hermes.
