# Programmatic Navigation in React Native

---

## üîπ What is Programmatic Navigation?

**Programmatic navigation** means navigating via code logic instead of a button press or user interaction.
Instead of `<Link/>` or `<TouchableOpacity/>`, you trigger navigation based on events like API success, validation, authentication, timers, etc.

### For example:

- Redirect user after login success
- Navigate after fetching data
- Navigate when onboarding is completed

---

## üîπ How Navigation Works in React Native

React Native commonly uses **React Navigation**.

To navigate programmatically, you use the **navigation object**, which gives methods such as:

| Method                  | Purpose                                                |
| ----------------------- | ------------------------------------------------------ |
| `navigation.navigate()` | Go to a new screen (can revisit previous ones)         |
| `navigation.push()`     | Push same screen multiple times (like stack entries)   |
| `navigation.replace()`  | Replace current screen (cannot go back)                |
| `navigation.goBack()`   | Go to previous screen                                  |
| `navigation.popToTop()` | Go to first screen in stack                            |
| `navigation.reset()`    | Reset entire navigation history (useful for auth flow) |

## üìå Programmatic Navigation When navigation is Not Available

Sometimes we are outside screen components (e.g., Redux action, API function).
Then we use a **Navigation Container Ref**.

### Create navigation ref

```javascript
import { createNavigationContainerRef } from '@react-navigation/native';

export const navigationRef = createNavigationContainerRef();

export function navigate(name, params) {
  if (navigationRef.isReady()) {
    navigationRef.navigate(name, params);
  }
}
```

### Wrap NavigationContainer

```javascript
<NavigationContainer ref={navigationRef}>
  <StackNavigator />
</NavigationContainer>
```

### Navigate from anywhere in code!

```javascript
import { navigate } from './RootNavigation';

navigate('Home', { loggedIn: true });
```

## üß† Interview Tips

In an interview you can summarize like this:

> **Programmatic navigation** allows navigating in React Native without user interaction.
> It is useful during conditional flows like login success, API completion, or onboarding.
> Using React Navigation, we call `navigation.navigate`, `replace`, or `reset`.
> If outside a screen, we use a **navigation ref** to access navigation globally.
