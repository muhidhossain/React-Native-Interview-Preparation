## What is EAS Build?

**EAS Build (Expo Application Services)** is the cloud build system for Expo and React Native apps. It allows you to build APK/AAB/IPA files **without needing Xcode or Android Studio locally**.

EAS uses a file called **`eas.json`** to define build profiles and configurations.

---

## âœ… 1. What is eas.json?

`eas.json` is a config file that tells EAS **how to build your app**.

You can define **multiple build profiles**, such as:

- `development`
- `preview`
- `production`

Each profile can specify:

- build type (apk, aab, simulator build)
- release channel or update channel
- environment variables
- credentials settings
- runtime versions
- Gradle/Xcode settings
