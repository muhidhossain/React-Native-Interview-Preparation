# Styling with StyleSheet and Flexbox

In React Native, styling is done with JavaScript objects that closely resemble CSS but are not fully CSS.
React Native provides the **StyleSheet API**, which helps create optimized, immutable style objects.

---

## âœ… Styling with StyleSheet in React Native

### Why use StyleSheet?

- **Performance optimization**: Styles created via `StyleSheet.create` are validated and cached, improving runtime performance.
- **Code organization**: Centralized, reusable styles keep components clean.
- **Autocompletion & error-checking** in editors.

### Example:

```jsx
import { StyleSheet, Text, View } from 'react-native';

export default function App() {
  return (
    <View style={styles.container}>
      <Text style={styles.title}>Hello React Native</Text>
    </View>
  );
}

const styles = StyleSheet.create({
  container: {
    flex: 1,
    backgroundColor: '#f2f2f2',
    alignItems: 'center',
    justifyContent: 'center',
  },
  title: {
    fontSize: 24,
    fontWeight: 'bold',
    color: '#333',
  },
});
```

### Key Features of StyleSheet

| Feature                                 | Explanation                                                       |
| --------------------------------------- | ----------------------------------------------------------------- |
| `StyleSheet.create()`                   | Adds validation + improves performance                            |
| No cascading                            | RN styles do not cascade like CSS                                 |
| Unitless                                | Values like `width: 100` assume "dp" (density-independent pixels) |
| Shorthand properties mostly unsupported | e.g., you can't use `margin: 10 5`                                |

## ðŸ§± Flexbox in React Native

React Native uses **Flexbox** by default for layout, but with some differences from browser Flexbox.

ðŸ‘‰ **Main goal**: Build responsive UI that scales across screen sizes.

### ðŸ“Œ Important Differences from Web Flexbox

**1. Default `flexDirection = "column"`** (on the web it's `"row"`)

This affects how elements stack.

**2. Percentage (%) units not commonly used**

Flexbox properties handle most responsive design.

---

## ðŸ”¥ Core Flexbox Concepts

### 1. flex

Determines how much space an element takes relative to siblings.

```jsx
flex: 1; // takes equal space
flex: 2; // takes double the space of flex:1
```

### 2. flexDirection

Defines main axis direction.

| Value            | Items arranged       |
| ---------------- | -------------------- |
| `row`            | horizontally         |
| `column`         | vertically (default) |
| `row-reverse`    | reverse horizontal   |
| `column-reverse` | reverse vertical     |

### 3. justifyContent

Aligns items along the **main axis**.

**Common values:**

- `flex-start`
- `center`
- `space-between`
- `space-around`
- `space-evenly`

### 4. alignItems

Aligns items perpendicular to the main axis.

**Values:**

- `flex-start`
- `center`
- `flex-end`
- `stretch` (default for height/width is not set)

### 5. alignSelf

Overrides `alignItems` for an individual child.

### 6. flexWrap

Allows wrapping items in multiple rows or columns.

## Flexbox Example:

```jsx
<View style={styles.container}>
  <View style={styles.box1} />
  <View style={styles.box2} />
  <View style={styles.box3} />
</View>;

const styles = StyleSheet.create({
  container: {
    flexDirection: 'row',
    justifyContent: 'space-between',
    padding: 20,
  },
  box1: { width: 50, height: 50, backgroundColor: 'red' },
  box2: { width: 50, height: 50, backgroundColor: 'green' },
  box3: { width: 50, height: 50, backgroundColor: 'blue' },
});
```

## ðŸŽ¯ Common Interview Questions with Answers

### 1. What is StyleSheet in React Native and why use it?

**Answer:**  
StyleSheet organizes styles in a centralized, optimized way. `StyleSheet.create` validates and caches the styles to improve performance and structure.

### 2. What are the differences between styling in React Native vs CSS?

- No cascading or inheritance
- Units are unitless
- Many CSS properties missing
- Shorthands not fully supported
- Flexbox is the main layout system

### 3. What is the default flexDirection in React Native?

`column` (unlike the web, which is `row`)

### 4. What does flex: 1 mean?

It tells the component to expand and fill available space proportionally.

### 5. Difference between justifyContent vs alignItems?

- `justifyContent` = main axis
- `alignItems` = cross axis

### 6. When do you use alignSelf?

When a single child needs different alignment than the group.

### 7. Can you use percentages in React Native?

Yes, but rarely recommended. Flexbox is preferred for responsive design.

### 8. What is the difference between flex-start and flex-end?

- `flex-start`: items start at beginning of axis
- `flex-end`: items pushed to end of axis
