## ðŸ”µ What is Redux?

**Redux** is a state management library that helps manage global and shared application state in a predictable way.

### Key Concepts

| Concept       | Meaning                                                       |
| ------------- | ------------------------------------------------------------- |
| **Store**     | A single object that holds the global state of your app       |
| **Action**    | Plain JS object that describes what happened                  |
| **Reducer**   | A pure function that takes state + action â†’ returns new state |
| **Dispatch**  | Sends actions to reducers to update state                     |
| **Subscribe** | Listens for state changes                                     |

### Why Redux is used in React Native?

Because mobile apps often have:
âœ” deeply nested components  
âœ” shared data across screens (auth, theme, user profile, cart, settings etc.)  
âœ” a need for predictable debugging

Redux ensures state changes are **traceable** and easier to maintain at scale.

## Code Comparison (Very Important for Interviews)

### ðŸ”´ Traditional Redux (long & noisy)

```javascript
// actions.js
export const INCREMENT = 'INCREMENT';
export const increment = () => ({ type: INCREMENT });

// reducer.js
const initialState = { count: 0 };
export default function counterReducer(state = initialState, action) {
  switch (action.type) {
    case INCREMENT:
      return { ...state, count: state.count + 1 };
    default:
      return state;
  }
}
```

### ðŸŸ¢ Redux Toolkit (short, clean)

```javascript
import { createSlice } from '@reduxjs/toolkit';

const counterSlice = createSlice({
  name: 'counter',
  initialState: { count: 0 },
  reducers: {
    increment: (state) => {
      state.count += 1;
    }, // direct mutation allowed via Immer
  },
});

export const { increment } = counterSlice.actions;
export default counterSlice.reducer;
```

## Using Redux Toolkit in React Native

### Store Setup

```javascript
import { configureStore } from '@reduxjs/toolkit';
import counterReducer from './counterSlice';

export const store = configureStore({
  reducer: {
    counter: counterReducer,
  },
});
```

### Provide Store to App

```javascript
import { Provider } from 'react-redux';
import { store } from './store';

export default function App() {
  return (
    <Provider store={store}>
      <YourNavigation />
    </Provider>
  );
}
```

### Access in Components

```javascript
import { useSelector, useDispatch } from 'react-redux';
import { increment } from './counterSlice';

const CounterScreen = () => {
  const count = useSelector((state) => state.counter.count);
  const dispatch = useDispatch();

  return <Button title="Add" onPress={() => dispatch(increment())} />;
};
```
