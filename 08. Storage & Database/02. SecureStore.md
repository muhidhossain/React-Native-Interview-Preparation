AsyncStorage stores data in **plaintext**, which is not secure for sensitive information.

**SecureStore:**

- Encrypts data automatically
- Uses native secure storage APIs
- Protects against data leaks if the device is compromised

## âœ… How SecureStore works internally

SecureStore uses:

- **Keychain** on iOS
